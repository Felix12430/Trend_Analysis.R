# Drought Trend Analysis in R

## Overview
R script for comprehensive time series analysis of drought indices using statistical methods.

## Analysis Methods
- Mann-Kendall test for monotonic trends
- Sen's slope estimator for trend magnitude
- Composite index creation (SPI, VCI, TCI)
- Seasonal variability analysis

## Outputs
- Time series plots
- Trend analysis reports
- Statistical significance testing
- Visualization charts

## Requirements
- R 4.0+
- Packages: readr, dplyr, lubridate, Kendall, zyp, ggplot2
